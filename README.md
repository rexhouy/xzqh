# Xzqh

Short description and motivation.

## Usage

javascript:
xzqh.open(["云南省", "昆明市", "五华区"], function(d) {console.log(d)} )

## Installation

```ruby
gem 'xzqh'
```

And then execute:

```bash
$ bundle
```

Or install it yourself as:

```bash
$ gem install xzqh
```
